# 智能网址导航系统 - 项目需求文档

## 📋 项目概述

### 项目名称
智能网址导航系统 (Smart Navigation System)

### 项目描述
基于Supabase和Vue框架开发的现代化网址导航系统，提供简洁的UI界面和实用的工具功能。系统主要实现网页书签的可视化管理和可拖拽分类，同时提供热门网址推荐和用户自定义书签管理功能。

### 技术栈
- **前端框架**: Vue 3 + Vite
- **后端服务**: Supabase (PostgreSQL + 实时功能)
- **UI组件**: 自定义组件库
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **样式**: CSS3 + 响应式设计

## 🎯 核心功能需求

### 1. 用户认证系统
- **用户注册/登录**: 支持邮箱注册和第三方登录
- **用户资料管理**: 头像上传、个人信息编辑
- **权限控制**: 基于角色的访问控制

### 2. 网址导航核心功能

#### 2.1 书签管理
- **可视化书签展示**: 卡片式布局，支持图标和缩略图
- **拖拽分类**: 支持书签在不同分类间拖拽移动
- **分类管理**: 创建、编辑、删除自定义分类
- **书签导入/导出**: 支持主流浏览器书签格式

#### 2.2 热门网址推荐
- **精选网址**: 预设热门网站分类（新闻、工具、娱乐等）
- **一键添加**: 用户可快速将推荐网址添加到个人收藏
- **智能推荐**: 基于用户行为推荐相关网址

#### 2.3 个人书签管理
- **自定义书签**: 用户可添加、编辑、删除个人书签
- **文件夹管理**: 支持多级文件夹结构
- **搜索功能**: 快速搜索书签和分类
- **标签系统**: 为书签添加标签便于分类

### 3. 实用工具集成
- **计算器**: 基础数学计算功能
- **日历组件**: 日期查看和事件管理
- **其他工具**: 根据用户需求扩展

## 🔧 技术实现方案

### 数据库设计
```sql
-- 用户表
users (
  id, email, username, avatar_url, created_at, updated_at
)

-- 用户资料表
user_profiles (
  id, user_id, display_name, bio, preferences
)

-- 分类表
categories (
  id, user_id, name, color, icon, parent_id, sort_order
)

-- 书签表
bookmarks (
  id, user_id, category_id, title, url, description, 
  favicon_url, created_at, updated_at
)

-- 好友关系表
friendships (
  id, user_id, friend_id, status, created_at
)

-- 共享文件夹表
shared_folders (
  id, name, owner_id, collaborator_id, created_at
)

-- 聊天消息表
messages (
  id, sender_id, receiver_id, content, type, created_at
)
```

### API接口设计
- **认证接口**: `/auth/*`
- **书签管理**: `/api/bookmarks/*`
- **分类管理**: `/api/categories/*`
- **好友管理**: `/api/friends/*`
- **协作功能**: `/api/collaboration/*`
- **聊天功能**: `/api/chat/*`

### 实时功能
- **实时同步**: 使用Supabase Realtime
- **WebSocket连接**: 聊天和协作功能
- **离线支持**: 本地存储和同步机制
